# AeroSpace Cheat Sheet

## How it works

**Workspaces** are virtual desktops numbered 1–9, all on the ultrawide. One visible at a time. Windows are explicitly assigned to workspaces.

**Tiling** is automatic within a workspace. One window fills the screen. Two windows split it. Each new window splits the remaining space. Orientation is automatic based on aspect ratio.

**To control splits**, use move (`Alt+Shift+hjkl`) to rearrange windows — left/right for horizontal splits, up/down for vertical.

---

## Workspaces

| Key | Action |
|-----|--------|
| `Alt+1`…`Alt+9` | Switch to workspace N |
| `Alt+Shift+1`…`Alt+Shift+9` | Move focused window to workspace N |
| `Alt+Tab` | Jump back to previous workspace |

---

## Focus & Move

| Key | Action |
|-----|--------|
| `Alt+H/J/K/L` | Focus window ← ↓ ↑ → |
| `Alt+Shift+H/J/K/L` | Move window ← ↓ ↑ → |

---

## Layout

| Key | Action |
|-----|--------|
| `Alt+T` | Toggle tiles ↔ accordion |
| `Alt+Shift+Space` | Toggle float / tile for focused window |
| `Alt+F` | Fullscreen (stays in tiling tree) |

---

## Resize

| Key | Action |
|-----|--------|
| `Alt+R` | Enter resize mode |
| `H/J/K/L` | Shrink/grow (50px) |
| `Shift+H/J/K/L` | Shrink/grow (150px) |
| `Esc` or `Enter` | Exit resize mode |

---

## Other

| Key | Action |
|-----|--------|

| `Alt+Shift+R` | Reload config |
